## The Three-Question Test

Before automating any step, answer these three questions. If you can't answer "yes" to all three, the step should stay manual or have a human checkpoint.

### Question 1: Are the possible actions limited?

Can you enumerate everything the agent might do at this step? If the action space is open-ended, automation is risky.
